Detailed Description

This static class offers an interface to serialize/deserialize data types (described by a Typecode) using CDR protocol inside a eProsima::FastBuffer.

void DynamicFastBuffers::SerializerAPI::deserialize ( void *  data,
Bytecode *  bytecode,
eProsima::FastCdr *  cdr 
)
static

This function deserializes any data defined by user into the buffer inside FastCdr object, using the rest of the functions implemented within this API class. User does not have to worry about the data type, because it is the bytecode that defines the kind of data the user wants to serialize.

Parameters
cdrA pointer to a FastCdr object containing a FastBuffer from where data will be deserialized
dataA void pointer to the data type by user where data will be deserialized.
Exceptions
NotEnoughMemoryExceptionThis exception is thrown when trying to deserialize in a position that exceed the internal memory size.
BadParamExceptionThis exception is thrown trying to deserialize in an invalid value.
